Introduction

This is the API-Documentation of the ALPHA2 version of mcstats.sol4it.de. Feel free to help developing this tool at GitHub: MinecraftPEStats

This Service is in an early ALPHA state! Everything could change at any time. Please be aware of this fact when using this service!
With mcstats.sol4it.de you can track servers and players. The following documentation is designed to teach you about how to do this.

Insert a Server

This Request allows you to add a new server to the tracking system. Your server will be queried every 5 minutes.
When your server were offline for over 24 Hours, your server will be disabled! In order to re-enable it again, you can call this request again.

Request   POST   http://mcstats.sol4it.de/api/insertServer 
{
    "serverHost":"your.server.host",
    "serverPort":19132
}
Response 
{
    "status": "OK",
    "serverId": "5n6y2fl4jeucdz3v",
    "error": false,
    "generated": "2018-03-16T19:34:23.378Z"
}

Query an existing Server

You can query a Server, which has been previously added to the tracking system. Just provide it's trackingId to get (filtered) Server information.
When your Server is already added to the system, you should always prefer this method, becaus it does cost way less performance, and It helps to keep your server actively tracked!

Banners

Get server Banner

This method returns a PNG image, based on recent queries. You should prefer this method, because it uses less performance.

Request   GET   http://mcstats.sol4it.de/banner/{serverId}.png
EMPTY
Response

Get server Banner by Ip and Port

This method returns a PNG image. This method fetches the server query in realtime, which can cause a slight delay. You should prefer the above method, because this might get removed when abused!
