Introduction

This document is designed to teach you how to easily integrate the Chart, and the Player's played time table into your website.
You must know a bit about JavaScript, CSS and HTML in order to use this methods. A pre-generated website you can just link to will follow later.
Please note, that we might change any of this features anytimes. So please check this page here regulary for updates.

1. Embedding the Javascript file

In order to use the functions down below, you need to include the following JavaScript file to your website. Replace {serverId} with the Id you've got when registering your server.

Including the js file 
<script type="text/javascript" src="https://mcstats.sol4it.de/embed/{serverID}.js"></script>

2. Using the Table generator

You can generate a Table with players ranked by their online time count in a give period. You can specify the time period as the second argument to the function.
The first argument has to be the table element or the Id of the target table element you want to render the player list in.
The second argument should be a valid timespan indicator like "hour","day","week","month"
The Third argument specifies the maximum limit which should be displayed. It defaults to 10.
The fourth and fifth arguments can be used to define functions for formatting data.
The library comes with default formatting functions shipped already, so you don't need to use this. If you don't want to use it, you can just pass null.
The first function gets the minute count of the current player passed. It should return a formatted string to be displayed in the table.
The second function gets the lastOnline date together with the selected timespan passed. It should return a formatted string aswell.
Last but not least, you cann pass a sixth argument, which should be a function to be called after rendering is done.

Please note: The raw table looks kinda ugly. It is meant to be styled by you. It got serveral classes to be adjusted in css.

Using the Table generation 
    mcstats.makeTable("myCoolTableElement","week",5,null,null,function(){
        console.log("Yeah my Table is ready :-)");
    });

3. Using the chart generator

You can generate a chart to display online players over time.
The function requires 3 arguments at max.
The first argument should be the target SVG element, or the ID of the SVG element you want to render the chart in.
The second argument should be the timespan you want to display. Eg: "hour","day","week","month".
The third argument can be a function to be called once the chart is rendered.

Please note: The function will include third party libraries like d3.js and nv.js (hosted on cloudflare cdn) to your websites head.

Using the Chart generation 
    mcstats.makeChart("myCoolSVGElement","week"function(){
        console.log("Yeah my Chart is ready :-)");
    });
